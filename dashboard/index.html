Recalls by cause

Cause is derived from free-text reason_for_recall via a keyword rule set (see README) — "Other" is the heuristic's residual, not a real category.

Multi-recall events

One real-world recall event can trigger many recall filings — different firms, lots, or product lines tied to the same incident.

Resolution time by severity
